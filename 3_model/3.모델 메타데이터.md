### 입력 데이터 스펙
- 입력 특성 수: 수치/주기형 9개 + 범주형 `category_group` 1개 (총 10개 입력 필드)
- 필수 전처리: StandardScaler , One-Hot Encoding , Median Imputation
- 특성 목록: entry_rank_log, T0_view_log, T0_engagement_ratio_log, latency_to_trend_log, pretrend_view_velocity_log, weekday_sin, weekday_cos, hour_sin, hour_cos, category_group

### 예측값 해석
| 출력값 | 의미 |
|--------|------|
| 0 | 유튜브 트렌딩의 성과 일반 |
| 1 | 유튜브 트렌딩의 성과 우수 |

### 모델 로드 및 예측 예시
```python
import joblib
import pandas as pd

# 모델 경로 설정
model_pipeline = joblib.load('backend/models/weighted_soft_voting.joblib')

# 신규 영상 데이터 입력 (T0 시점 기준 10개 피처)
new_video = pd.DataFrame([{
    "entry_rank_log": 2.5,
    "T0_view_log": 12.3,
    "T0_engagement_ratio_log": 0.05,
    "latency_to_trend_log": 1.2,
    "pretrend_view_velocity_log": 4.5,
    "weekday_sin": 0.974,
    "weekday_cos": -0.222,
    "hour_sin": 0.5,
    "hour_cos": 0.866,
    "category_group": "Entertainment"
}])

# 모델 예측
prediction = model_pipeline.predict(new_video)[0]
probability = model_pipeline.predict_proba(new_video)[0, 1]

print(f"트렌딩 성과 예측: {'우수(1)' if prediction==1 else '일반(0)'}")
print(f"우수 성과일 확률: {probability:.2%}")
```
